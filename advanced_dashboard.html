واجهة API

نقاط API المتاحة:

  • GET /api/factories - حالة المصانع
  • GET /api/products - قائمة المنتجات
  • GET /api/exports - الملفات المصدرة
  • POST /api/factories/start - بدء تشغيل مصنع
  • POST /api/export - تصدير المنتجات

اختبار API:

curl http://localhost:8000/api/factories
